## 如何填写

bootstrap 任务会从项目代码中自动提取规范。你也可以手动更新：
1. 记录项目的**实际约定**（不是理想状态）
2. 包含项目代码库中的**代码示例**
3. 列出**禁止的模式**及原因
4. 添加团队踩过的**常见错误**
